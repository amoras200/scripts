# Repositorio de armazenamento de scripts.

## 📂 Scripts Disponíveis:

### 🌦️ clima.sh
Script que utiliza o comando curl para consumir a API do wttr.in de clima, . 

Funcionalidade: Exibe as condições climáticas atuais de São Paulo diretamente no terminal.

### 💾 backup.sh
Utilizado para automação de cópias de segurança de diretórios importantes.

Funcionalidade: Cria cópias de segurança datadas.

### 📦 compactador.sh
Script para compressão de arquivos de forma rápida.

Funcionalidade: Facilita o empacotamento de logs ou arquivos de configuração.
